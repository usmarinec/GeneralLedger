Suggested Workflow

  1. Create or select an accounting entity.
  2. Create or select a fiscal year.
  3. Create at least two accounts, such as Cash and Service Revenue.
  4. Create a balanced STANDARD journal entry.
  5. Post the journal entry.
  6. Generate the unadjusted trial balance.
